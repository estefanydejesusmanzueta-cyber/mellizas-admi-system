##  Definición de la Empresa
**Papelería Las Mellizas** es un negocio dedicado a la venta de productos escolares, de oficina y artículos diversos de papelería.  
Este proyecto será aplicado dentro de esta empresa como referencia para analizar y mejorar sus procesos administrativos y operativos.

---

##  Problema Identificado
Actualmente, los procesos administrativos de la papelería se realizan de forma manual, lo que puede provocar:

- Errores en el registro de ventas e inventario.
- Pérdida o desorganización de información.
- Retrasos en la búsqueda de productos, clientes o historial de ventas.
- Falta de control sobre usuarios y acceso a datos.
- Dificultad para tomar decisiones por falta de información centralizada y actualizada.

---

## Solución Propuesta
**Mellizas Admin System** es un sistema web interno diseñado para **automatizar y optimizar** la gestión administrativa y operativa de la Papelería Las Mellizas.

El sistema permitirá:

- Administrar productos y categorías.
- Controlar inventario en tiempo real.
- Registrar clientes.
- Registrar ventas y consultar historial de transacciones.
- Gestionar usuarios con roles y permisos.
- Centralizar información para una gestión eficiente, segura y rápida.

Este sistema será de uso exclusivo del personal autorizado, eliminando la necesidad de registros manuales.

---

##  Objetivos del Sistema

### Objetivo General
Desarrollar una aplicación web que permita administrar eficientemente productos, inventario, ventas, clientes y usuarios, optimizando los procesos administrativos de la papelería.

### Objetivos Específicos
- Desarrollar un módulo para la gestión de productos.
- Implementar control de inventario actualizado.
- Permitir registro y administración de clientes.
- Implementar un módulo de ventas con historial de transacciones.
- Permitir autenticación de usuarios con roles y permisos.
- Garantizar una interfaz intuitiva y fácil de usar.

---

##  Alcance del Proyecto
El sistema cubrirá las siguientes funcionalidades:

- Gestión de productos y categorías.
- Control de inventario.
- Gestión de clientes.
- Registro de ventas e historial de transacciones.
- Control de usuarios con diferentes roles.
- Almacenamiento de datos en base de datos.
- Generación de registros e historiales para consultas internas.

---

## Límites y Delimitación
- El sistema será **de uso interno** y **no estará disponible para clientes externos**.
- Solo será utilizado por personal autorizado.
- El proyecto se enfocará únicamente en la **gestión administrativa** de la papelería.
- No incluye tienda online ni compras por internet.
- No incluye integración con pagos electrónicos (a menos que se agregue en futuras versiones).

##  Requerimientos Funcionales
El sistema debe permitir:

1. **Gestión de productos**
   - Registrar productos
   - Editar productos
   - Eliminar productos
   - Consultar listado de productos

2. **Gestión de inventario**
   - Actualizar stock de productos
   - Visualizar existencias disponibles
   - Alertas o control de productos con bajo inventario (opcional)

3. **Gestión de clientes**
   - Registrar clientes
   - Editar clientes
   - Consultar información de clientes
   - Eliminar clientes (según permisos)

4. **Registro de ventas**
   - Registrar ventas
   - Asociar ventas a clientes (si aplica)
   - Consultar historial de ventas
   - Consultar detalle de transacciones

5. **Gestión de usuarios**
   - Registro de usuarios (admin)
   - Autenticación de inicio de sesión
   - Control de roles y permisos
   - Restringir funciones según el rol

---

## Requerimientos No Funcionales
El sistema debe cumplir con:

- **Seguridad:** acceso mediante autenticación y roles.
- **Usabilidad:** interfaz intuitiva y fácil de usar.
- **Rendimiento:** respuestas rápidas al consultar productos, ventas y clientes.
- **Fiabilidad:** almacenamiento correcto y consistente en base de datos.
- **Disponibilidad:** acceso desde navegador dentro del entorno interno.
- **Escalabilidad:** permitir agregar módulos o funciones en el futuro.
- **Mantenibilidad:** código organizado y fácil de mejorar o corregir.
